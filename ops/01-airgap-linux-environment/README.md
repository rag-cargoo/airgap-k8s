# 01 Airgap Linux Environment

- 과제 항목: 인터넷이 불가능한 네트워크 환경, 리눅스 기본 구성
- 현업 기준: 폐쇄망 인프라는 제공된 상태로 보고, 네트워크 차단과 OS 기본 상태를 확인한다.
- 실습 기준: 필요 시 AWS Terraform 또는 VM 방식으로 폐쇄망 환경을 재현한다.
- 실제 설치 절차는 VM 노드 기준으로 이어간다.

## Directories
- `aws-terraform-simulation/`: AWS 기반 폐쇄망 시뮬레이션 인프라
- `vm-airgap-setup/`: VM 기반 폐쇄망 구성 설명과 설정
- `../../assets/offline-assets/`: 폐쇄망 설치 준비용 작업 원본 자산 저장소
