# 03-01. 공통 사전점검 및 반입

## 대응 경로
- `ops/03-kubernetes-cluster/common/`
- `ops/03-kubernetes-cluster/common/01-access-and-transfer/`
- `ops/03-kubernetes-cluster/common/02-preflight/`

## 실행 순서
```bash
source ops/01-airgap-linux-environment/scripts/load-project-env.sh
make ops-runtime-bundle-run
make ops-runtime-bundle-verify
make 03-01-preflight-run
make 03-01-preflight-verify
```

설명: bastion 실행 원본 번들을 준비하고, master/worker에 `kubeadm-preflight-check.sh`를 업로드한 뒤 원격 사전점검을 실행하고 재검증한다.

## 검증 기준
```text
[RESULT] SUCCESS
```

설명: `03-01-preflight-run`과 `03-01-preflight-verify`가 모두 `SUCCESS`여야 다음 단계로 진행한다.

## 관련 스크립트
- `ops/03-kubernetes-cluster/common/scripts/kubeadm-preflight-check.sh`
- `ops/03-kubernetes-cluster/common/scripts/run-preflight-stage.sh`
- `ops/03-kubernetes-cluster/common/scripts/verify-preflight-stage.sh`
