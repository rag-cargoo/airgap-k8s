# 00-01. 과제 원문

## 서버 구축 테스트

## 🎈문제 1

## 주제
- 온프레미스(폐쇄망 = Air-gapped) Kubernetes 및 모니터링 서비스 구축 및 활용

## 내용
1. 인터넷이 불가능한 네트워크 환경
2. 리눅스 기본 구성
  - 사용자 및 네트워크 설정
    - `devops`라는 사용자를 생성하고 sudo 권한 부여
    - 호스트 이름을 `k8s-master`, `k8s-worker1`로 설정
    - `/etc/hosts`에 각 노드 IP와 이름을 등록
3. Kubernetes 클러스터 구성
  - master + worker 구성
  - 네트워크 플러그인: Calico 또는 Flannel
4. 서비스 배포 및 모니터링 설정 능력
  - MySQL(or MariaDB)
  - MongoDB
  - Prometheus
  - Grafana
  - Grafana Alloy
5. Prometheus + Grafana 외부 접근 설정

## 제출
- 서버 구축 과정에 대한 상세한 단계별 매뉴얼 작성본 (자유양식: word/hwp/ppt 등)
- 추가적으로 서버 구축 설정에 사용된 파일 압축(ZIP, 7z 등등) 첨부
