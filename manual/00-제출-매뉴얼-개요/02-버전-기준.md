# 00-02. 버전 기준

## Kubernetes
- Version: `v1.35.3`
- `kubeadm`: `v1.35.3`
- `kubelet`: `v1.35.3`
- `kubectl`: `v1.35.3`

## Runtime
- 기본 container runtime은 `containerd`로 한다.
- Kubernetes 공식 `Container Runtimes` 문서 기준의 CRI 호환 구성을 따른다.

## CNI
- 과제 요구사항에 따라 `Calico` 또는 `Flannel` 중 하나를 선택한다.
- 최종 선택한 CNI manifest만 제출 압축본에 포함한다.

## Version Rule
- control plane, kubelet, kubectl 버전은 `v1.35.3` 기준으로 맞춘다.
- 오프라인 이미지 목록은 `kubeadm config images list --kubernetes-version v1.35.3` 결과를 기준으로 산정한다.
